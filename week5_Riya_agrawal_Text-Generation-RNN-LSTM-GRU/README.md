Text Generation using RNN, LSTM, and GRU

Overview
This project demonstrates text generation using three recurrent neural network architectures:
- Vanilla RNN
- LSTM
- GRU
The models are trained on a custom AI and Machine Learning corpus to learn sequential patterns and generate coherent text.

Features
- Text preprocessing and tokenization
- Sequence generation
- Embedding layer
- Vanilla RNN implementation
- LSTM implementation
- GRU implementation
- Training loss comparison
- Text generation

Technologies Used
- Python
- TensorFlow / Keras
- NumPy
- Matplotlib

Learning Outcomes
- Understanding recurrent neural networks
- Comparing RNN, LSTM, and GRU
- Sequence modeling
- Natural Language Processing fundamentals
