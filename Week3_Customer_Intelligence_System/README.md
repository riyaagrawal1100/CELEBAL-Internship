Customer Intelligence System using Classification, Ensemble Learning & Clustering

Project Overview-
This project develops a Customer Intelligence System using machine learning techniques, including clustering, classification, and ensemble learning.
The objective is to identify meaningful customer segments and build predictive models classifying future observations into those segments.
The project uses country socio-economic data as customer profiles and performs intelligent segmentation based on multiple development indicators.

Key Insights
  Child mortality was the most influential feature.
  GDP and income strongly impacted segmentation.
  Life expectancy and fertility rate contributed significantly to cluster formation.
  DBSCAN successfully identified socio-economic outliers.

Conclusion

The Customer Intelligence System successfully combines clustering, classification, and ensemble learning techniques to generate actionable insights and predictive intelligence. The framework can support segmentation, resource allocation, and strategic decision-making.
